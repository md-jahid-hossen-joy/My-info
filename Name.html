1. Data Types: Text & Math

Computers need to know what kind of information they are looking at. We call these Data Types.

The two most common types are Strings (text wrapped in quotes) and Integers (whole numbers used for math).

# This is a String (Text)
name = "Pizza"

# This is an Integer (Whole Number)
slices = 8

2. Data Types: Decimals & Truth

What if you want to store a number with a decimal? In Python, we call that a Float (floating-point number).

We also have Booleans, which are super simple: they can only be True or False. They are like light switches!

# This is a Float (Decimal)
price = 19.99

# This is a Boolean (True/False)
is_tasty = True

3. Leaving Comments

Sometimes you want to leave notes in your code for humans to read, but you want the computer to ignore them. These are called Comments.

In Python, just type the hash symbol #. Anything after the hash on that line becomes invisible to the computer!

# This is a comment. The computer ignores it.
score = 100 # You can put them after code, too!

print("Hello!") # Only "Hello!" gets printed.

4. Type Conversion (Shape Shifting!)

Sometimes, data is in the wrong format. For example, the text "5" is a String, not a number. You can't do math with text!

We can use built-in magic spells to convert them: int() turns things into integers, and str() turns things into text.

fake_number = "5" # It's trapped in quotes!
real_number = int(fake_number) # Now it's an integer

print(real_number + 10) # Output: 15

5. Why do we convert?

Imagine you ask a user for their age using Python's input() feature. User input always comes back as a String (Text).

If you want to calculate how old they will be next year, you must convert their text answer into a math number first!

age_text = "20" # Imagine a user typed this

# We shape-shift it into a Float (Decimal)
age_number = float(age_text)
print(age_number) # Output: 20.0
